Project Overview - 
This project is a fully functional Kanban Task Management Board built using Vanilla JavaScript, HTML, and CSS.
It allows users to create, manage, drag, and organize tasks across different stages of work — To Do, In Progress, and Done — with persistent storage using LocalStorage.
The project demonstrates real-world frontend development concepts, clean DOM manipulation, and state management without using any external libraries or frameworks.

Features
Add new tasks with title and description
Drag & Drop tasks between columns
Delete tasks instantly
Dynamic task count per column
Persistent data using LocalStorage
Clean and reusable JavaScript functions
Responsive and user-friendly UI
No frameworks — pure JavaScript logic

Tech Stack
HTML5 – Semantic structure
CSS3 – Layout & styling
JavaScript (ES6) – Core logic, DOM manipulation

LocalStorage API – Data persistence
